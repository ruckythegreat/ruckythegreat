## 🧠 About Me

Hey! I'm **Rucky** — a developer who loves building things from scratch and figuring out how stuff works under the hood.

- 🎮 Built a **Unity 2D Roguelike** game from the ground up
- 📱 Shipped a **Flutter mobile app** (Uangky — personal finance tracker)
- 🔍 Done **APK reverse engineering** for fun and learning
- 🧩 Created **Rucky.js** — a lightweight homegrown JS framework
